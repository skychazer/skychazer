I'm a data engineer at Data Mavericks. Most of my work is building AI agents, RAG systems, and text-to-SQL tools, along with the Snowflake and AWS pipelines that keep them fed with real data. I usually end up working on both sides of that, the data plumbing underneath and the AI layer on top, which is a mix I don't run into all that often.

Lately most of my time goes into making these systems reliable enough to actually run in production, not just look good in a demo.

## Things I've built

**[Decck](https://decck.alinverma.com)** turns messy source material (transcripts, spreadsheets, notes) into slide decks you build by chatting with it. You bring your own API key (Claude, OpenAI, Gemini, or OpenRouter), and it handles brand styling, lets you edit individual fields on a slide without spending an LLM call, and exports to PDF. Built with Next.js.
