## Things I've built

<a href="https://decck.alinverma.com" target="_blank" rel="noopener noreferrer"><strong>Decck</strong></a> turns messy source material (transcripts, spreadsheets, notes) into slide decks you build by chatting with it. You bring your own API key (Claude, OpenAI, Gemini, or OpenRouter), and it handles brand styling, lets you edit individual fields on a slide without spending an LLM call, and exports to PDF. Built with Next.js.
